#California House Price Prediction API

Ushbu loyiha ma'lum bir hududdagi uylarning xususiyatlariga (xonalar soni, aholi daromadi, joylashuvi va h.k.) qarab, uylarning taxminiy o'rtacha (median) narxini bashorat qilib beruvchi **Machine Learning API** hisoblanadi.

Loyihada **Linear Regression** modelidan foydalanilgan hamda API qismi **FastAPI** orqali
